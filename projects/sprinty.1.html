Sprinty

Sprinty is an online tool that helps students run design sprints.

Try it out at sprinty.co⤳

THE STORY

Design sprints are an exceptional way to collaboratively produce design outcomes in short bursts of time.

Sprinters divide their time into fractions to spend on each stage of the design process. For example, given an hour, sprinters spend 10 minutes unpacking the problem, 10 minutes researching solutions, and so on.

THE CHALLENGE

Direct observation and participation showed that students found it challenging to focus on producing design while simultaneously tracking and managing time and tasks on the sprint.

WHAT DOES THE USER NEED?

  • A tool that helps them know what tasks to do and in what order
  • A tool to help them manage how long they spend on each task
  • A tool that is simple, flexible, and is going to be used in a hurry

The solution: a webapp that guides the user through each stage of the sprint process.

Each stage is timed proportionally to how long the user has to run the sprint.

Sprinty is flexible and scales to any display. The interface is designed to be used in a rush.

Putting sprinty together

Research involved taking part in, and observing how people worked in sprints.

Challenges included doing the math to make sure timers worked.

Early prototypes were created before any styling.
